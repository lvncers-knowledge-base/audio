# DAC

「Digital to Analog Converter」の略で、デジタル信号をアナログ信号に変換する機器です。

- 役割: PCやスマホ、ゲーム機などの中にある「0と1」のデジタル信号を、人間が聴けるアナログ信号に変換します。
- なぜ必要か: スピーカーやイヤホンはアナログ信号でしか動かないため、変換を行わないと音が出ません。

## DACの主要スペックと読み方

| スペック | 意味 | 目安 |
| - | - |
| THD+N（全高調波歪み率） | 音の純粋さ | 低いほど良い、-100dB以下が優秀 |
| SNR（S/N比） | ノイズの少なさ | 高いほど良い、120dB超が優秀 |
| ダイナミックレンジ | 最小〜最大音量の差 | 広いほど良い |
| 出力インピーダンス | ヘッドホンとのマッチング | 低いほど良い |

### THD+N（全高調波歪み率）

音の波形って本来こうあるべき

```txt
理想の波形：  ～～～～～～
実際の波形：  ～～～～～～ + おまけの歪み成分
```

この「おまけ」が高調波歪み。
元の音にない周波数成分が混入するから、音が濁ったり、長時間聴いてると疲れる原因になる。

- -80dB以下：普通に聴く分には気にならない
- -100dB以下：かなり優秀、音の透明感が出る
- -120dB以下：フラグシップレベル

### SNR（S/N比）

```txt
      出したい音（Signal）
SNR = ──────────────────
      背景のノイズ（Noise）
```

これが低いと、静かな場面で「サー」っていうホワイトノイズが聴こえてくる。
HD600みたいな高感度ヘッドホンだとノイズが乗りやすいから、SNRは特に重要。

- 100dB：及第点
- 120dB超：ほぼノイズ聴こえない
- 130dB超：業務グレード

### 出力インピーダンス

さっきのダンピングファクターの話に繋がるんだけど、電気的にいうとヘッドホンと出力インピーダンスで分圧が起きてるんだよ。

```txt
出力インピーダンス：10Ω
ヘッドホン：300Ω（HD600）

→ 分圧でほとんど問題ない ✅

出力インピーダンス：100Ω
ヘッドホン：32Ω（普通のイヤホン）

→ 電圧が大幅にロスする ❌ 音がスカスカになる
```

**目安は「ヘッドホンのインピーダンスの1/8以下」** にするのがオーディオ業界の経験則（1/8ルール）。

### ダイナミックレンジ

```txt
pppピアニッシモ ←────────────── 130dB ──────────────→ fffフォルティッシモ
```

これが狭いと、静かな音を上げたとき先にノイズフロアが上がってきちゃう。
オーケストラとかジャズで「空気感」を感じたいなら広いほど良い。
